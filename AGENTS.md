# Сбор данных о сайтах

Go-проект для анализа доменов: парсит title, description, h1, текст, контакты (телефоны, email, ИНН, соцсети, адрес). Результаты в SQLite.

## Команды

```bash
go run main.go           # запуск (указать файл с доменами)
go build                 # сборка
go test                  # тесты
```

## Структура

- `internal/` - основная логика
- `cmd/` - точка входа
- `web/` - фронтенд (HTML + Tailwind, порт 1439)
- `logs/` - логи

## Настройки

Многопоточность, задержки, переобход - в конфиге или флагах CLI.

## Важно

- Неудачные загрузки тоже сохраняются в БД
- Логирование всех действий в отдельную папку
